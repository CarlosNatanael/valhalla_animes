Sobre o Projeto Valhalla Animes

Uma jornada de aprendizado com Python, Flask e paixão por animes.

Nossa Missão

O Valhalla Animes nasceu como um projeto de demonstração com o objetivo de criar uma plataforma web moderna e funcional. A missão foi praticar e aprofundar conhecimentos em desenvolvimento web, desde o backend com Python e Flask até a criação de uma interface de usuário responsiva e atraente.

Tecnologias Utilizadas

Este site foi construído com um conjunto de ferramentas modernas e poderosas:

Python
Flask
SQLAlchemy
HTML5
CSS3
JavaScript
